# 🛍️ LuxeCart - Premium E-Commerce Website

A modern, responsive, and feature-rich **Premium E-Commerce Website** built using **React + Vite**. This project demonstrates modern frontend development practices, reusable component architecture, responsive UI design, and an excellent user experience.

> 🚀 Developed as a portfolio project to showcase React.js frontend development skills.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/luxecart.git
```

## Navigate

```bash
cd luxecart
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

---

# 📦 Build

```bash
npm run build
```
